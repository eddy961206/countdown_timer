Privacy Policy

This Privacy Policy explains the types of personal information that Chrome Timer & Alarm Extension (hereinafter referred to as "this program") collects, the purpose of collection, how it is used, and how it is protected.

1. Types of Information Collected

This program only stores the following information locally to provide timer and alarm functionality:

  • Timer duration set by users
  • Alarm times set by users

This information is stored locally on your device and is not transmitted to any external servers.

2. Purpose of Collection

The stored information is used solely to:

  • Maintain timer functionality while the browser is running
  • Ensure alarm notifications are triggered at the correct time
  • Preserve user settings between browser sessions

3. Data Protection

All data is stored locally using Chrome's built-in storage APIs. No data is transmitted to external servers, ensuring complete privacy for all users.

4. Chrome Permissions

This program requires the following Chrome permissions:

  • Storage: To save timer and alarm settings
  • Notifications: To display timer and alarm alerts
  • Alarms: To trigger notifications at specified times

5. Changes to the Privacy Policy

This Privacy Policy may be updated as necessary, and any changes will be announced through the Chrome Web Store listing.
